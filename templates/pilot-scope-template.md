## 7. 용어 정의 (Glossary)

| 용어 | 정의 |
|------|------|
| 파일럿 (Pilot) | 전사 확산 전에 소규모로 먼저 검증하는 단계 |
| KPI | Key Performance Indicator, 핵심 성과 지표 |
| 베이스라인 모델 | 비교 기준이 되는 초기 단순 모델 |
| 드리프트 (Data Drift) | 시간이 지남에 따라 실제 데이터 분포가 학습 데이터와 달라지는 현상 |
| 혼동 행렬 (Confusion Matrix) | 분류 모델의 예측 결과를 정답과 비교한 표 |
| 재현율 (Recall) | 실제 불량 중 모델이 맞게 불량으로 예측한 비율 (놓침 방지 중요) |
| 정밀도 (Precision) | 모델이 불량으로 예측한 것 중 실제 불량인 비율 (오경보 방지 중요) |
| XAI | Explainable AI, 설명 가능한 인공지능 |
| 엣지 디바이스 | 현장 설비 근처에서 직접 AI 추론을 수행하는 소형 컴퓨팅 장치 |
