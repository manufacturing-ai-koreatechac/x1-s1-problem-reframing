# X1-S1: 문제재정의 워크숍

> **[v1.7]** KOREATECH 제조AI 실무 교육과정 — X1 시작 세션 (이론 2h + 실습 2h = 4h)

## 학습 목표

- 현장 문제를 AI 해결 가능한 형태로 재정의 (5 Whys · PDCA)
- 제조 AI 4가지 데이터 형태 분류 워크시트 작성
- AI 적합성 판단 + Tool 설계 체크리스트 완성
- Track A/B → X1 연결 구조 이해 (Agentic AI 도입 방향 설정)

## 실습 구성 (Lab 3종)

| 순서 | 내용 | 도구 |
|:----:|------|------|
| Lab 1 | 현장 문제 재정의 (5 Whys) | 업무-AI 매핑 워크시트 |
| Lab 2 | 제조 현장 데이터 4형태 분류 워크시트 | `worksheets/business-ai-mapping.csv` |
| Lab 3 | AI 적합성 판단 + Tool 설계 체크리스트 | `worksheets/ai-feasibility-checklist.md` |

## 제공 자료

- `worksheets/business-ai-mapping.csv` — 업무-AI 매핑 워크시트
